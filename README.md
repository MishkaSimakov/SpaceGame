# Веб-версия игры "Космические баталии"

## Контейнеры

Проект можно удобно запустить, используя docker compose. Для этого достаточно выполнить:
```shell
docker compose up
```

Структура контейнеров:
1. База данных. Используется postgres, который запускается в отдельном контейнере. Остальные контейнеры могут взаимодействовать с ним по сети.
2. Сервер. Доступен на порту 3000.
3. adminer. Доступен на порту 8080. Через него можно удобно администрировать базу данных.
